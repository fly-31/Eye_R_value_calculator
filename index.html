👁️ Eye Tracking — MG Screening

Upload eye-tracking recordings; the tool flags patients whose eyes fail to follow the laser or fatigue over time. Screening aid only.

ℹ️ How to use / required data format
  1. Your files must match the column format in this template: Example Google Sheet.
  2. Each recording is a .csv with columns Time(sec), LH, RH, LV, RV, TargetH, TargetV.
  3. Upload the individual .csv files or a .zip containing them (one zip per patient works well).
  4. Horizontal uses LH/RH vs TargetH; vertical uses LV/RV vs TargetV.

⚠️ Needs valid Target (laser) columns. Screening signal, not a diagnosis.
