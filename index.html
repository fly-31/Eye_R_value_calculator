👁️ Eye Movement R² Calculator

Upload eye-tracking recordings and download a spreadsheet of R² values (how strongly each eye's position correlates with time) for every category. Everything runs in your browser — nothing is uploaded to a server. WARNING: Website does not allow .zip file format - extract to .csv and drag+drop

ℹ️ How to use / required data format
  1. Your files must match the column format in this template: Example Google Sheet.
  2. Each recording is a .csv with columns Time(sec), LH, RH, LV, RV, TargetH, TargetV.
  3. Upload the individual .csv files or a .zip containing them (one zip per patient works well).
  4. Patient name and category (Horizontal/Vertical, frequency, B/R) are read automatically from the file names — keep the original names.

⚠️ The calculator will not work if the files are not in the correct format. Only the LH and RH channels are used (this matches the reference sheet).
